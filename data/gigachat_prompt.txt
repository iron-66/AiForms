Проанализируй следующий текст и извлеки все вопросы, на которые может ответить человек. Выведи вопросы в виде массива объектов JSON. JSON должен иметь ключ "pages", который содержит массив вопросов, и каждый вопрос должен иметь следующие ключи:
- "id": Уникальный идентификатор вопроса, как показано в форме.
- "question_text": Точный текст вопроса, как он указан в форме.
- "hint_text": Любая подсказка или дополнительная информация для вопроса, или null, если подсказки нет.
- "answer_type": Тип ответа, ожидаемый для вопроса. Возможные значения: "number", "email", "phone_number", "date", "selection", "text". Если тип вопроса
- "options": Содержит варианты ответов для вопросов типа "selection" 

Вот пример структуры JSON вашего ответа:

{
  "pages": [
    {
      "id": 1,
      "question_text": "example",
      "hint_text": "example",
      "answer_type": "selection",
      "options": ["option 1", "option 2", "option 3", "option 4"]
    },
  ]
}

Примечания:
- Извлекай только те вопросы, которые явно указаны в исходном тексте.
- Не генерируй вопросы, которых нет в форме.
- Если вопрос разделён на несколько частей или имеет несколько ответов, включи каждую часть или ответ как отдельный вопрос в JSON.
- Убедись, что значения "id" соответствуют идентификаторам, как они указаны в форме.
- Язык - английский или русский.
- Не пиши заметки о своих результатах, только массив JSON!

Проанализируй предоставленный текст и создай массив объектов JSON, содержащий все вопросы из формы с указанной структурой.
